## Installation

```R
# First install the R package "devtools" if not installed
devtools::install_github('marzione00/COVID_19_HACK')
```

## Usage

Load the package

```R
require(covid19)
```

The function of the package is `runC19()` and is run without arguments.

```R
runC19()
```
